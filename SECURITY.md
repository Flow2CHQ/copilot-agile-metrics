# Security Policy

## Supported Versions

This project contains **GitHub Copilot prompt files** — Markdown text files with no executable code, no dependencies, and no server infrastructure.

There is no software to patch or version to "support" in the traditional sense. Security concerns in this project are therefore limited to:

- Prompt injection risks (a contributed prompt that attempts to manipulate Copilot into harmful behaviour)
- Privacy concerns (a prompt that inadvertently leaks data or instructs Copilot to send data to third parties)
- Misleading or deceptive prompt content

## Reporting a Vulnerability

If you discover a security concern in this project — including a prompt that could be used to manipulate AI behaviour in harmful ways — please **do not open a public GitHub issue**.

Instead, report it privately by emailing: **security@flow2c.com**

Please include:
- Which file(s) are affected
- A description of the concern and potential impact
- Steps to reproduce or demonstrate the issue (if applicable)

You can expect an acknowledgement within **5 business days** and a resolution or public disclosure decision within **30 days**.

## Scope

This policy covers the contents of this repository only. It does not cover GitHub Copilot itself, the GitHub platform, or any third-party services. For vulnerabilities in those products, please refer to their respective security policies.
